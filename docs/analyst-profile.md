## Key Design Principles (from Maya's experience)
1. **Traders scan, they don't read.** Headlines > articles. Scores > paragraphs.
2. **Time is the x-axis of everything.** Every insight should map to a time window on the chart.
3. **Magnitude matters more than direction.** Traders know bullish/bearish — they need to know HOW MUCH.
4. **Source credibility is implicit.** Reuters > random blog, but don't make the user think about it — weight it automatically.
5. **The best insight is the one you didn't have to ask for.** Push, don't pull.
